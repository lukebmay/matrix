# Third-party fonts

## Matrix Code / Matrix Resurrected

- **Source:** [Rezmason/matrix](https://github.com/Rezmason/matrix)
- **Files:** `Matrix-Code.ttf`, `Matrix-Resurrected.ttf`
- **License:** MIT (see `Rezmason-matrix-MIT.txt`)
- **Copyright:** Copyright (c) 2018 Rezmason

Used for digital-rain noise glyphs (classic film code set).
`Matrix-Resurrected.ttf` is vendored for completeness; rain uses classic.

## Ubuntu Sans Mono

- **Source:** [canonical/Ubuntu-Sans-Mono-fonts](https://github.com/canonical/Ubuntu-Sans-Mono-fonts)
  (same family as Ghostty / nvim: “UbuntuSansMono Nerd Font”, without Nerd
  icon patches)
- **Files:** `UbuntuSansMono-Bold.ttf` (wght=700 instance)
- **License:** Ubuntu Font Licence 1.0 (see `UbuntuSansMono-UFL.txt`)
- **Copyright:** Canonical Ltd

English monospaced face for settled card/saying text. Bold weight so stem
thickness sits closer to Matrix Code rain glyphs; terminal/computer feel.

## Courier Prime

- **Files:** `CourierPrime-*.ttf`
- **Role:** Fallback stack only
